# python-boilerplate

## Usage

### Setup repository

```
git clone https://github.com/Ancient-Scapes/python-boilerplate.git <folder_name>
git remote set-url origin https://github.com/<username>/<repository>.git
```

### Edit requirements.txt

```
beautifulsoup4
lxml
requests
selenium
```

### Windows

```
./setup.bat
```

### Mac

```
./setup.sh
```

### Hello World

```
python ./main.py
```
